# 指南

ViewerPro 是一个现代化图片预览组件，支持缩放、拖拽、旋转、全屏、下载、缩略图导航、自定义渲染和信息面板。

## 推荐阅读顺序

- [快速开始](/guide/getting-started)
- [安装](/guide/installation)
- [配置实验台](/api/options)
- [ViewerPro API](/api/viewer-pro)
- [类型定义](/api/types)

## 核心能力

- 图片预览：打开、关闭、切换、缩放、拖拽、旋转。
- 缩略图导航：打开预览后按需渲染，缩略图懒加载。
- 自定义能力：支持自定义 loading、渲染节点和信息面板。
- 移动端：支持移动端工具栏和滑动切换。
